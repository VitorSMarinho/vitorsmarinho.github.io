Meus Projetos

  • Pipeline de Ingestão e Processamento

    Criação de um pipeline completo no Synapse integrando com Databricks para transformação e armazenamento otimizado em Data Lake.

  • Arquitetura para Data Platform

    Definição e implementação de uma arquitetura de dados escalável para analytics e reporting usando ferramentas Azure.

  • Processos de Ingestão e Governança de Dados

    Desenvolvimento de pipeline robusto para ingestão de arquivos JSON com validação de nomenclatura, cópia para Data Lake e tratamento automatizado de erros. Todo o fluxo foi documentado para garantir governança e rastreabilidade.

  • Pipeline de Transferência e Disponibilização de Tabelas no Data Lake

    Criação de processo automatizado para transferência de tabelas para o Data Lake, com logs detalhados e validação de integridade, garantindo a confiabilidade e disponibilidade para análises posteriores.

  • Gerenciamento Centralizado de Servidores com Azure Arc

    Implementação de arquitetura de gestão de servidores utilizando Azure Arc, entregando 100% de eficiência no monitoramento e controle remoto. A solução foi desenhada para ser replicável e escalável, tornando-se uma oferta estratégica para clientes da empresa.

  • Automação de Processamento de Arquivos CSV via Logic Apps

    Desenvolvimento de fluxo automatizado com Logic App para captura de arquivos CSV diretamente de e-mails, com filtros de nomenclatura e extensão. Arquivos são processados, validados e inseridos no banco de dados. Os registros são movidos para repositórios específicos conforme o status (sucesso, falha ou auditoria), garantindo conformidade e rastreabilidade completa.
